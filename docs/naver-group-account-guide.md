# 네이버 단체 계정 멤버 추가 가이드

## 개요

네이버 단체 아이디에서 하위 계정(멤버)을 추가하는 방법.
개발자 센터 앱 등록 시 여러 명이 관리해야 할 때 필요.

## 멤버 추가 경로

1. **단체 아이디 마스터 계정**으로 네이버 로그인
2. 우측 상단 프로필 → **내정보** 클릭
3. **보안설정** 탭 선택
4. **멤버 비밀번호** 섹션에서 **발급** 클릭

직접 링크: https://nid.naver.com/user2/help/myInfo

## 멤버 발급 절차

1. 멤버 이름(또는 호칭) 입력
2. 멤버가 사용할 이메일 주소 입력
3. **발급** 버튼 클릭
4. 해당 이메일로 "네이버 이용약관 동의" 요청 메일 발송됨
5. 멤버가 메일에서 동의하면 비밀번호 발급 완료

## 멤버 로그인 방법

- **아이디**: 단체 아이디 (마스터와 동일)
- **비밀번호**: 개별 발급받은 멤버 비밀번호

## 제한사항

- 최대 **10명**까지 멤버 추가 가능
- 동시 로그인 가능
- 마스터가 언제든 비밀번호 회수 가능

## 주의사항

- **개인 아이디**에는 이 메뉴가 없음 (단체 아이디만 가능)
- **멤버 계정**으로 로그인하면 관리 메뉴 안 보임 (마스터만 가능)
- 멤버 비밀번호는 마스터 비밀번호와 다름
