## 6. 손익분기점 (BEP) 분석

### 6-1. 월 고정비만 커버

```
월 고정비: ₩40,500 ($30)

Premium 1건 순수익 (혼합 모델 기준):
  매출:        ₩9,900
  이미지 원가: 8크레딧 × ₩146 = ₩1,168
  테마 원가:   2크레딧 × ₩61 (Sonnet) = ₩122
  순수익:      ₩8,610

BEP = ₩40,500 ÷ ₩8,610 = 4.7건

→ 월 Premium 5건 판매하면 고정비 커버
```

### 8-2. Opus가 정당화되는 조건

```
1. 품질 차이가 전환율에 미치는 영향:
   Opus 품질이 좋아서 전환율 2%p 상승 시 (8% → 10%):
   추가 매출: 2명 × ₩9,900 = +₩19,800
   추가 원가: 20cr × ₩304 = -₩6,080
   순 효과: +₩13,720 > 추가 Opus 비용 ₩24,300?
   → 전환율 5%p 이상 상승해야 정당화 ❌

2. safelist 실패율 개선:
   Sonnet safelist 실패율 20% → Opus 5% 라면:
   재생성 비용 절감: 15건 × ₩61 = ₩915
   vs Opus 추가 비용: 100건 × ₩243 = ₩24,300
   → 실패율 개선으로는 정당화 안 됨 ❌
```
