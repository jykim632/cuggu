## 1. 현재 구현 상태 (데모 가능)

### 완료된 핵심 기능

| 기능 | 상태 | 데모 포인트 |
|------|------|-------------|
| **카카오/네이버 로그인** | ✅ 완료 | 소셜 로그인 한 번에 가입 |
| **청첩장 CRUD** | ✅ 완료 | 생성/수정/삭제 가능 |
| **템플릿 4종** | ✅ 완료 | Classic, Modern, Minimal, Floral |
| **탭 기반 편집기** | ✅ 완료 | 기본정보/갤러리/계좌/AI사진 탭 |
| **AI 웨딩 사진 생성** | ✅ 완료 | **핵심 차별화 포인트** |
| **갤러리 업로드** | ✅ 완료 | S3 + CloudFront CDN |
| **계좌 정보 관리** | ✅ 완료 | 양가 계좌 등록 |
| **대시보드** | ✅ 완료 | 청첩장 관리, 통계 |
| **관리자 페이지** | ✅ 완료 | 사용자/청첩장 관리 |
| **공개 청첩장 뷰** | ✅ 완료 | `/inv/[id]` 페이지 |
| **섹션 순서 변경** | ✅ 완료 | 드래그로 섹션 재배치 |

### 데모 시나리오 (권장 순서)

1. **랜딩 페이지** → 가치 제안 확인
2. **카카오 로그인** → 원클릭 가입
3. **대시보드** → "새 청첩장 만들기" 클릭
4. **편집기 - 기본정보** → 신랑/신부 정보 입력
5. **편집기 - AI 사진** → ⭐ 증명사진 업로드 → 스타일 선택 → 생성
6. **편집기 - 갤러리** → 사진 업로드
7. **미리보기** → 실시간 반영 확인
8. **공개 URL** → 실제 청첩장 링크 공유

## 2. 미구현 기능 (MVP 필수)

### Priority 1 - 런칭 필수

| 기능 | 예상 공수 | 담당 | 비고 |
|------|----------|------|------|
| **RSVP 기능** | 2-3일 | 개발 | 참석 여부 + 인원수 + 축하 메시지 |
| **결제 시스템** | 3-4일 | 개발 | Toss Payments 연동 |
| **카카오톡 공유** | 1일 | 개발 | Open Graph + 카카오 SDK |
| **지도/길찾기** | 1-2일 | 개발 | Kakao Map API |

### Priority 2 - 런칭 후 빠르게

| 기능 | 예상 공수 | 담당 | 비고 |
|------|----------|------|------|
| **모바일 반응형 QA** | 2일 | QA/개발 | iOS Safari, 안드로이드 테스트 |
| **다양한 가족 형태** | 1일 | 개발 | 한부모/고인/익명 표기 |

| **보안 강화** | 2일 | 개발 | Rate limiting, 입력 검증 |

### Priority 3 - 향후 로드맵

| 기능 | 비고 |
|------|------|
| 드래그 앤 드롭 편집기 | dnd-kit, 복잡도 높음 |

| 카카오톡 알림톡 | RSVP 알림 |
| 축의금 송금 버튼 | Toss/카카오페이 |

## 4. 기술 부채 & 리스크

### 알려진 이슈

| 이슈 | 심각도 | 상태 |
|------|--------|------|
| NextAuth sessions 테이블 충돌 | 중간 | `cuggu-9xi` 이슈 등록됨 |
| AI 생성 실패 시 크레딧 환불 | 낮음 | 수동 처리 중 |
| 지도 미구현 | 높음 | 빈 섹션 표시 중 |

### 스케일링 고려사항

- **AI 비용**: 현재 Replicate 사용 ($0.025-0.04/이미지)
  - 월 1,000회 이상 시 self-hosted ComfyUI 검토
- **이미지 저장소**: S3 + CloudFront
  - 월 10GB 이상 시 비용 모니터링 필요
- **DB 성능**: Supabase 무료 티어
  - 사용자 1,000명 이상 시 유료 플랜 전환
