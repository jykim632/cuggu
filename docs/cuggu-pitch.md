# Cuggu — AI가 만드는 나만의 모바일 청첩장

---

## 한 줄 요약

**증명사진 한 장으로 웨딩 화보를, AI 한 마디로 청첩장 디자인을.**

---

## 문제

한국에서 매년 15만 쌍이 결혼한다. 이 중 모바일 청첩장을 사용하는 비율은 빠르게 증가하고 있지만, 기존 서비스는 **템플릿을 고르고 텍스트를 채우는** 수준에 머물러 있다.

| 기존 서비스의 한계 | 예비 부부의 현실 |
|-------------------|----------------|
| 100종 넘는 템플릿이지만 결국 비슷비슷 | "남들과 다른 청첩장"을 원하지만 디자인 역량 없음 |
| 웨딩 화보 별도 촬영 필요 (200~500만원) | 촬영 일정 잡기도 어려움 |
| 커스터마이징 한계 (색상/폰트 정도) | "우리만의 분위기"를 표현하고 싶음 |

---

## 솔루션

Cuggu는 **AI를 핵심에 놓은** 모바일 청첩장 플랫폼이다.

### 1. AI 웨딩 사진 생성

증명사진 1장 업로드 → AI가 웨딩 컨셉 사진 4장 생성

- 10가지 스타일: 클래식 스튜디오, 한복, 가든, 해변, 시네마틱 등
- 얼굴 보존 기술(PhotoMaker) 적용 — "이게 진짜 우리 사진 같다"
- 생성 시간: 10~30초
- **웨딩 촬영 200만원 vs Cuggu 1,000원**

### 2. AI 테마 생성

"벚꽃 느낌의 따뜻한 청첩장" 한 마디면 AI가 색상, 폰트, 레이아웃을 자동 생성

- 매번 구조적으로 다른 디자인 (갤러리 배치, 섹션 스타일, 간격까지)
- 빠른 생성(2초) / 정밀 생성(5초) 선택 가능
- 마음에 안 들면 재생성 — 10번 해도 200원

### 3. 완성형 청첩장 에디터

- 8탭 편집기: 기본정보, 예식장(카카오맵 연동), 인사말, 갤러리, 계좌, RSVP, 템플릿, 설정
- 실시간 미리보기 (디바이스별 아이폰/갤럭시)
- 2초 자동저장
- 발행 즉시 카카오톡/링크/네이티브 공유

## 시장

| 지표 | 수치 |
|------|------|
| 한국 연간 결혼 건수 | ~15만 건 |
| 모바일 청첩장 시장 규모 (추정) | 150~300억원/년 |
| 모바일 청첩장 침투율 | 빠르게 상승 중 |
| 주요 경쟁사 | 바른손카드, 모바일초대장 등 |

**시장 트렌드**: 종이 청첩장 → 모바일 전환 가속. MZ세대 커플의 "개성 표현" 수요 증가.

## 경쟁 우위

| | 바른손 (기존 강자) | Cuggu |
|---|---|---|
| 템플릿 | 100+종 (수작업 디자인) | 6종 + **AI 무한 생성** |
| 사진 | 사용자가 직접 업로드 | **AI가 웨딩 사진 생성** |
| 커스터마이징 | 색상/폰트 선택 | **자연어로 디자인 지시** |
| 차별화 | 다양한 선택지 | **유일한 AI 청첩장** |

핵심 포지셔닝: **"고르는 청첩장"에서 "만들어주는 청첩장"으로.**

## 비즈니스 모델

### 가격 정책

| 플랜 | 가격 | 포함 내용 |
|------|------|----------|
| 무료 | ₩0 | 청첩장 1개, AI 생성 2회, 기본 템플릿 |
| 프리미엄 | ₩9,900 (일회성) | AI 생성 10회, 프리미엄 템플릿, 워터마크 제거 |
| AI 크레딧 추가 | ₩1,000/회 | 단건 구매 |
| AI 크레딧 번들 | ₩8,000/10회 | 20% 할인 |

### 수익성

| AI 기능 | 원가/회 | 판매가 | 마진율 |
|---------|--------|--------|--------|
| 테마 생성 (Gemini Flash) | ~₩3 | ₩1,000 | **97%** |
| 테마 생성 (Claude Sonnet) | ~₩72 | ₩1,000 | **28%** |
| 사진 생성 (PhotoMaker) | ~₩113 | ₩1,000 | **89%** |
| 사진 생성 (Flux Pro) | ~₩221 | ₩1,000 | **78%** |

**가중 평균 마진: 72~89%.** AI 비용이 하락 추세이므로 마진은 시간이 갈수록 개선.

## 기술

| 영역 | 스택 |
|------|------|
| 프론트엔드 | Next.js 16, TypeScript, Tailwind CSS, Framer Motion, Zustand |
| 백엔드 | Next.js API Routes, NextAuth.js v5 (카카오 로그인) |
| DB | PostgreSQL (Supabase) + Drizzle ORM |
| AI (사진) | Replicate (Flux Pro, PhotoMaker), OpenAI (GPT Image), Google Gemini |
| AI (테마) | Anthropic Claude, OpenAI GPT-4o, Google Gemini |
| 스토리지 | AWS S3 + CloudFront |
| 분석 | PostHog |
| 배포 | Vercel |

**멀티 프로바이더 아키텍처**: AI 모델 6종을 팩토리 패턴으로 관리. 비용/품질에 따라 모델 전환 가능. 특정 벤더 종속 없음.

## 왜 지금인가

1. **AI 비용 급락**: Gemini Flash 테마 생성 1회 ₩3. 1년 전이라면 ₩300이었을 것
